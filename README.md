# Login & Registration_Dashboard

## Project Overview

This project is a simple **Login and Registration Dashboard** created using **HTML and CSS**. It demonstrates basic user interface design, form validation using HTML5, and page navigation.

The application consists of:

* A **Login Page** (`login.html`)
* A **Register Page** (`register.html`)
* A **Dashboard Page** (`dashboard.html`)
* A shared **stylesheet** (`style.css`)

This project is intended for academic purposes and front-end practice only. No backend or database is included.
---

### User Manual

### Login Page Instructions

* Enter your **Username** and **Password**.
* Click the **SIGN IN** button.
* If all required fields are filled, the user will proceed to the next page (demo redirect).

If the user **does not have an account**:

* Click the **"Register here"** link to go to the registration page.

---

### Registration Page Instructions

* Fill in the following required fields:

  * Full Name
  * Username
  * Email
  * Password
  * Confirm Password
* All fields are mandatory. The form will not submit if any field is empty.
* Click the **REGISTER** button.
* After successful validation, the user is redirected to the **Login Page**.

If the user already has an account:

* Click **"Login here"** to return to the login page.
