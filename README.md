SIMPLE LOGIN , REGISTER AND DASHBOARD (HTML & CSS)

This project is a simple SIMPLE LOGIN , REGISTER AND DASHBOARD created using HTML and CSS. It demonstrates basic user interface design, form validation, and page navigation. Also, this project is intended for academic purposes and front-end practice only. No backend or database is included.

The application consists of:

Login Page (`login.html`)

Register Page (`register.html`)

shared stylesheet (`style.css`)



**User Manual**

**Login Page Instructions**

Enter your Username and Password.
Click the **SIGN IN** button.
If all required fields are filled, the user will proceed to the next page which is the dashboard.

If the user **does not have an account**:

Click the **"Register here link to go to the registration page"**.

**Registration Page Instructions**

Fill in the following required fields:

   Full Name
   
   Username
   
   Email
   
   Password
   
   Confirm Password
   
All fields are mandatory. The form will not submit if any field is empty.

Click the **REGISTER** button.

After successful validation, the user is redirected to the Login Page.


Click **"Login here"** to return to the login page.
